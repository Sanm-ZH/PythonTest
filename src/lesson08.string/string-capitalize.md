#### 描述
Python capitalize()将字符串的第一个字母变成大写,其他字母变小写。
#### 语法
capitalize()方法语法：
```
str.capitalize()
```
#### 参数
无。
#### 返回值
该方法返回一个首字母大写的字符串。

#### 实例
以下实例展示了capitalize()方法的实例：
```python
#!/usr/bin/python3

str = "this is string example from github....wow!!!"

print ("str.capitalize() : ", str.capitalize())
```
以上实例输出结果如下：
```
str.capitalize() :  This is string example from github....wow!!!
```
